# Portfolio Análisis — Panel de Proyectos

## Resumen

Esta repo contiene una pequeña aplicación de administración de proyectos hecha con Streamlit (`app.py`). La app lee/escribe en una base de datos SQLite y usa funciones bajo la carpeta `function/` para crear, modificar, eliminar proyectos y para generar/subir la web del portfolio.
